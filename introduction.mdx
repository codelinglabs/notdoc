## Notilify API

Use the Notilify API to send SMS, manage API keys, handle webhooks, and monitor usage.

## Base URL

`https://api.notilify.com/v1`

## Authentication

Authenticate requests with a bearer token in the `Authorization` header.

```bash
Authorization: Bearer <token>
```

## Start Here

1. Sign in to get your auth token.
2. Create or retrieve your API key.
3. Send your first message with `POST /v1/message`.
4. Configure delivery webhooks to track delivery outcomes.

Use the API Reference section on the left for full endpoint details.
